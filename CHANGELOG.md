# Changelog

## 0.3.0 - 2026-06-15

- Added Cloudflare R2 video storage.
- Added a 5GB application storage cap with usage display.
- Added Cloudflare Pages Functions API for videos, streams, history, and agent polling.
- Added Windows startup task scripts for the local PC agent.
- Switched the frontend to cloud control mode for mobile use.
- Converted user-facing project files to English.

## 0.2.0 - 2026-06-15

- Improved video and history UI.
- Added local video upload and URL import.
- Changed FFmpeg default streaming mode to copy.

## 0.1.0 - 2026-06-15

- Created the initial StreamAgain-style control panel.
- Added local agent support for FFmpeg reruns.
- Added project documentation and Cloudflare Pages configuration.
